# IATRO Frontend

Modern Python GUI frontend for the pediatric diagnostic inference system.

## Features

- **Palantir-inspired Design**: Dark theme, clean interface, professional aesthetics
- **Interactive Diagnosis**: Real-time symptom selection and probability updates
- **Visual Feedback**: Progress indicators, confidence metrics, and diagnosis rankings
- **New Diagnosis**: Quick reset to start fresh

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```

Make sure `pediatric.db` is in the parent directory or specify path with `--db` argument.
