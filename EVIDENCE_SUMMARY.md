# Evidence Rows Summary - Pediatric.db

## Overview

**Total Evidence Rows: 1,139**

All evidence linking diseases to symptoms (phenotypes) with diagnostic parameters and citations.

## Files Created

1. **`ALL_EVIDENCE.txt`** - Human-readable formatted text file (11,107 lines)
   - Easy to read and search
   - Includes all citation details
   - Formatted for easy scanning

2. **`ALL_EVIDENCE.csv`** - Spreadsheet format (1,140 rows including header)
   - Can be opened in Excel, Google Sheets, etc.
   - Easy to filter and sort
   - All columns included

## What's Included in Each Evidence Row

### Diagnostic Parameters
- **LR+** (Positive Likelihood Ratio)
- **LR-** (Negative Likelihood Ratio)
- **Sensitivity**
- **Specificity**
- **Prevalence** (when available)

### Source Information
- **PMID** (PubMed ID)
- **Guideline Organization** (AAP, IDSA, WHO, etc.)
- **Year** of publication
- **Study Design** (Guideline, Systematic Review, etc.)
- **Evidence Grade** (A, B, C)

### Citation Details
- **Authors**
- **Journal**
- **Volume** and **Issue**
- **DOI**
- **URL**
- **Page**, **Table**, **Figure**, **Section** references
- **Full Reference** (complete citation)
- **Data Location** (specific location in source)

### Context Information
- **Age Range** (min/max months)
- **Setting** (ED, primary care, inpatient, etc.)
- **Region** (geographic context)
- **Extraction Date** (when data was extracted)
- **Notes** (additional clinical notes)
